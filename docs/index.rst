Features
--------

* AI 기반 대화형 인터페이스
* MCP (Model Context Protocol) 지원
* 다양한 LLM 프로바이더 지원
* 웹훅 및 GitHub 통합
* 모듈식 아키텍처

Quick Start
-----------

.. code-block:: bash

   pip install -e .
   python app.py
